
# 냉장고 관리

객체지향 프로그래밍(OOP) 패러다임을 적용하여 구현한 [냉장고 관리] 서비스입니다. TypeScript와 React 기반의 객체지향 아키텍처를 설계하여 컴포넌트의 재사용성과 확장성을 높였다.

---

## 개발 환경 및 사용 기술
- **Language**: TypeScript (v5.x)
- **Framework/Library**: React (v18.x), Next.js (선택사항인 경우 기술)
- **Architecture**: Object-Oriented Component Design (객체지향 컴포넌트 설계)

---

## 프로젝트 구조 (Architecture)
객체지향 원칙을 준수하기 위해 다음과 같이 구조화되어 관리한다.
- `src/app/`: 전체 애플리케이션의 핵심 로직 및 페이지 라우팅 관리
- `src/components/`: 재사용 가능한 객체 지향적 UI 컴포넌트 모듈
- `src/styles/`: 독립적인 스타일 속성 관리
